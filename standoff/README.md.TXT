# Stand Off

Stand Off is a wild-west themed recreation of the 'James Bond' hand game.

## Purpose of Project

The purpose of this project is to show off an example of socket programming and 
multi-threading with the use of sound effects and a user interface.

## Version

Version Number: 2.0.1

## How to Start This Project

If you are in the project files, run the main method on the StandOff class, 
otherwise just double click the jar file.
